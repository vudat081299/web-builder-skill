Design tokens

Tất cả màu đến từ CSS variables. Đổi token → đổi toàn hệ. Đây là các token hay dùng nhất (xem đủ trong cashy-ui.css). Bảng dưới đổi theo Light/Dark bạn đang bật.

Nhóm token chính

Tham chiếu các token “semantic” này thay vì thang xám thô, để cả 2 theme luôn đúng.

/* Bề mặt & chữ */
--cash-canvas      /* nền app          */   --cash-fg         /* chữ chính     */
--cash-surface     /* card, table      */   --cash-fg-muted   /* chữ phụ       */
--cash-surface-2   /* header, sọc      */   --cash-fg-subtle  /* chữ mờ nhất   */
--cash-border      /* viền hairline    */   --cash-border-strong

/* Màu có ý nghĩa — mỗi màu có 3 biến */
--cash-success / --cash-success-soft / --cash-success-text   /* xanh: đã trả  */
--cash-danger  / --cash-danger-soft  / --cash-danger-text    /* đỏ:  quá hạn  */
--cash-warning / --cash-warning-soft / --cash-warning-text   /* vàng: sắp hạn */
--cash-info    / --cash-info-soft    / --cash-info-text      /* xanh dương    */

/* Hình khối & đổ bóng */
--cash-radius-sm | --cash-radius | --cash-radius-lg | --cash-radius-pill
--cash-shadow-sm (nghỉ) | --cash-shadow-md (nổi)

/* Font — đổi 1 dòng này là đổi toàn bộ chữ (xem trang Fonts) */
--cash-font
💡
Cần một màu mới? Thêm token ở cả :root và .dark, rồi dùng token — đừng bao giờ viết mã hex thẳng vào component.
