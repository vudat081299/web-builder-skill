Nhập liệu

Text input

Ô nhập trung tính; màu duy nhất là trạng thái lỗi (đỏ). Gói trong .cash-field (label + control + help/error). Ghép prefix / suffix bằng .cash-input-group — addon là chữ (₫, %, https://) hoặc icon.

Cơ bản

Tối đa 120 ký tự. 
<div class="cash-field">
  <label class="cash-label">Tên giao dịch</label>
  <input class="cash-input" placeholder="VD: Ăn trưa…" />
  <span class="cash-help">Tối đa 120 ký tự.</span>
</div>

Prefix & suffix (input-group)

Addon đứng trước (prefix) hoặc sau (suffix) ô nhập — cho tiền tệ, %, đường dẫn, tìm kiếm…

https://
search
₫ .00
% / năm 
<!-- prefix: chữ -->
<div class="cash-input-group">
  <span class="cash-input-group__addon">https://</span>
  <input class="cash-input" value="cashy.app/vi" />
</div>

<!-- prefix: icon -->
<div class="cash-input-group">
  <span class="cash-input-group__addon"><span class="cash-ico cash-ico--sm">search</span></span>
  <input class="cash-input" placeholder="Tìm giao dịch…" />
</div>

<!-- prefix + suffix -->
<div class="cash-input-group">
  <span class="cash-input-group__addon">₫</span>
  <input class="cash-input" value="1.280.000" />
  <span class="cash-input-group__addon">.00</span>
</div>

Trạng thái

Lỗi dùng state .is-invalid (không phải --invalid) — thống nhất với các state runtime khác (.is-active, .is-open). Kèm aria-invalid cho screen reader.

Email không hợp lệ. 
<input class="cash-input is-invalid" value="sai-email" aria-invalid="true" />
<span class="cash-error">Email không hợp lệ.</span>

<input class="cash-input" value="TXN-2026-0714" disabled />

Chọn màu — cash-color

Ô chọn màu gốc được dọn sạch: cả ô là một swatch bo góc mang đúng màu đang chọn (bỏ viền/đệm mặc định của trình duyệt, không còn "ô vuông trong nút tròn"). Ghép nhãn hex bằng .cash-color-field. Dùng cho màu danh mục / nhãn.

#6366F1 
<span class="cash-color-field">
  <input class="cash-color" type="color" value="#6366f1" /><code>#6366F1</code>
</span>
<input class="cash-color cash-color--pill" type="color" />   <!-- tròn -->
<input class="cash-color cash-color--sm" type="color" />     <!-- nhỏ -->
