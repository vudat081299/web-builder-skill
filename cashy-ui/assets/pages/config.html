Config / Tweak

Bảng tinh chỉnh trượt ra từ bên phải (nút tune trên thanh trên cùng). Kéo các thông số và xem giao diện đổi ngay lập tức. Việc chỉnh ở đây chỉ ảnh hưởng bản docs đang mở — hoàn toàn không đụng tới cashy-ui.css hay component gốc. Chỉnh xong bấm Xuất .md để lấy khối token, rồi dán vào source (hoặc đưa cho AI cập nhật).

info
Cơ chế: bảng chỉ ghi các biến CSS (--cash-*) lên :root của trang docs. Vì component dùng biến, chúng đổi theo tức thì; còn file thư viện thì bất biến. Màu áp dụng cho theme đang xem — muốn chỉnh cả sáng & tối thì đổi theme rồi xuất thêm một lần nữa.
