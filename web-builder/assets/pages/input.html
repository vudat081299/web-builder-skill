Nhập liệu

Text input

Ô nhập trung tính; màu duy nhất là trạng thái lỗi (đỏ). Gói trong .wb-field (label + control + help/error). Ghép prefix / suffix bằng .wb-input-group — addon là chữ (₫, %, https://) hoặc icon.

Cơ bản

Tối đa 120 ký tự. 
<div class="wb-field">
  <label class="wb-label">Tên giao dịch</label>
  <input class="wb-input" placeholder="VD: Ăn trưa…" />
  <span class="wb-help">Tối đa 120 ký tự.</span>
</div>

Prefix & suffix (input-group)

Addon đứng trước (prefix) hoặc sau (suffix) ô nhập — cho tiền tệ, %, đường dẫn, tìm kiếm…

https://
search
₫ .00
% / năm 
<!-- prefix: chữ -->
<div class="wb-input-group">
  <span class="wb-input-group__addon">https://</span>
  <input class="wb-input" value="ledger.app/vi" />
</div>

<!-- prefix: icon -->
<div class="wb-input-group">
  <span class="wb-input-group__addon"><span class="wb-ico wb-ico--sm">search</span></span>
  <input class="wb-input" placeholder="Tìm giao dịch…" />
</div>

<!-- prefix + suffix -->
<div class="wb-input-group">
  <span class="wb-input-group__addon">₫</span>
  <input class="wb-input" value="1.280.000" />
  <span class="wb-input-group__addon">.00</span>
</div>

Trạng thái

Lỗi dùng state .is-invalid (không phải --invalid) — thống nhất với các state runtime khác (.is-active, .is-open). Kèm aria-invalid cho screen reader.

Email không hợp lệ. 
<input class="wb-input is-invalid" value="sai-email" aria-invalid="true" />
<span class="wb-error">Email không hợp lệ.</span>

<input class="wb-input" value="TXN-2026-0714" disabled />

Chọn màu — wb-color

Ô chọn màu gốc được dọn sạch: cả ô là một swatch bo góc mang đúng màu đang chọn (bỏ viền/đệm mặc định của trình duyệt, không còn "ô vuông trong nút tròn"). Ghép nhãn hex bằng .wb-color-field. Dùng cho màu danh mục / nhãn.

#6366F1 
<span class="wb-color-field">
  <input class="wb-color" type="color" value="#6366f1" /><code>#6366F1</code>
</span>
<input class="wb-color wb-color--pill" type="color" />   <!-- tròn -->
<input class="wb-color wb-color--sm" type="color" />     <!-- nhỏ -->

Bảng màu — wb-swatches

Picker tối giản, thuần CSS: một lưới ô màu preset để chọn từ — giữ lựa chọn trong các hue đã duyệt (đúng thang màu), thay vì mở hộp chọn màu của hệ điều hành. Đặt hue mỗi ô bằng --wb-swatch-color; ô đang chọn thêm .is-selected (viền nhấn theo màu chữ, đọc rõ trên mọi hue). Biến thể --round (tròn) · --sm (nhỏ). Cần màu tự do thì ghép thêm .wb-color ở trên.

<div class="wb-swatches" role="group" aria-label="Chọn màu">
  <button type="button" class="wb-swatch is-selected"
          style="--wb-swatch-color:var(--wb-chart-1)" aria-label="Chàm"></button>
  <button type="button" class="wb-swatch"
          style="--wb-swatch-color:var(--wb-chart-2)" aria-label="Teal"></button>
  <!-- … -->
</div>

<!-- tròn + nhỏ -->
<div class="wb-swatches wb-swatches--round wb-swatches--sm">…</div>

Bộ chọn màu — wb-colorpicker

Panel custom thay hẳn hộp thoại màu của hệ điều hành (thứ bung ra khi bấm wb-color) — tối giản, đúng tông: vùng SV (đậm/sáng) + thanh hue cầu vồng + ô nhập hex + hàng preset (tái dùng wb-swatches). Thư viện lo giao diện; phần kéo-chọn + quy đổi HSV do một handler nhỏ đảm nhiệm (bản docs này có sẵn) hoặc một lib headless. Kéo thử vùng màu / thanh hue, gõ hex, hay bấm preset:

# 
<div class="wb-colorpicker">
  <div class="wb-colorpicker__area"><span class="wb-colorpicker__thumb"></span></div>
  <div class="wb-colorpicker__hue"><span class="wb-colorpicker__thumb"></span></div>
  <div class="wb-colorpicker__foot">
    <span class="wb-colorpicker__preview"></span>
    <div class="wb-input-group">
      <span class="wb-input-group__addon">#</span>
      <input class="wb-input" value="6366F1" />
    </div>
  </div>
  <div class="wb-swatches wb-swatches--sm">…</div>   <!-- preset -->
</div>
